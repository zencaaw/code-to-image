# code-to-image

## Exemples d'entrée pour le/les code(s)

### Un code
- **12345**

### Plusieurs codes
- Si vous sélectionnez "," dans les séparations : **123,456,789**

- Si vous sélectionnez "/" dans les séparations : **123/456/789**
